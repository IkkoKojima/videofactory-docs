1. VideoFactory について

VideoFactory は、開発者(小島一航 / Ikko Kojima)が自身の YouTube チャンネルに動画をアップロードするために個人的に使用するコマンドラインツールです。一般には配布・提供していません。ユーザーは開発者本人のみです。

2. YouTube API サービスの利用

VideoFactory は YouTube API サービス(YouTube API Services)を利用しています。本ツールを利用することにより、ユーザーは以下に同意したものとみなされます。

  • YouTube 利用規約: https://www.youtube.com/t/terms
  • Google プライバシーポリシー: https://policies.google.com/privacy

3. アクセス・収集するデータ

  • 本ツールは Google OAuth 2.0 を通じて、認可したアカウント自身の YouTube チャンネルへの動画アップロード権限(スコープ: https://www.googleapis.com/auth/youtube.upload)のみを取得します。
  • 本ツールが YouTube に送信するのは、開発者が制作した動画ファイルと、そのタイトル・説明文・タグ・サムネイル等のメタデータのみです。
  • 本ツールは、視聴者・第三者のデータ、他のユーザーのチャンネルやコンテンツに関するデータには一切アクセスせず、収集・保存もしません。
  • Cookie、アクセス解析、広告技術は一切使用しません。

4. データの保存と共有

  • OAuth 認証トークンは、開発者のローカルPC上にのみ保存されます(外部サーバーには保存しません)。
  • 取得した認証情報およびアップロードに関する情報を、第三者に提供・販売・共有することはありません。
  • 外部へのデータ送信は、Google/YouTube の公式 API エンドポイントに対するもの以外に存在しません。

5. アクセス権の取り消しとデータの削除

  • ユーザー(開発者本人)は、Google のセキュリティ設定ページ https://security.google.com/settings/security/permissions からいつでも本ツールのアクセス権を取り消すことができます。
  • 本ツールが保存する認可済みデータは OAuth トークンのみです。ローカルに保存されたトークンファイルを削除することで、保存された認可済みデータは完全に削除されます。
  • 本ツールの利用を終了する場合、保存されたトークンは削除されます。トークン以外に YouTube API サービス経由で取得・保存しているデータはありません。

6. お問い合わせ

English

1. About VideoFactory

VideoFactory is a command-line tool used personally by its developer (Ikko Kojima) to upload videos to the developer's own YouTube channel. It is not distributed or offered to the public. The sole user is the developer.

2. Use of YouTube API Services

VideoFactory uses YouTube API Services. By using this tool, the user agrees to be bound by:

  • YouTube Terms of Service: https://www.youtube.com/t/terms
  • Google Privacy Policy: https://policies.google.com/privacy

3. Data Accessed and Collected

  • The tool obtains, via Google OAuth 2.0, only the permission to upload videos to the authorizing account's own YouTube channel (scope: https://www.googleapis.com/auth/youtube.upload).
  • The only data sent to YouTube are video files created by the developer and their metadata (title, description, tags, thumbnail).
  • The tool never accesses, collects, or stores data about viewers, third parties, or other users' channels or content.
  • No cookies, analytics, or advertising technologies are used.

4. Data Storage and Sharing

  • OAuth tokens are stored only on the developer's local machine (never on external servers).
  • No authorized data is sold, shared, or transferred to any third party.
  • The tool communicates only with official Google/YouTube API endpoints.

5. Revoking Access and Data Deletion

  • Access can be revoked at any time via Google security settings: https://security.google.com/settings/security/permissions.
  • The only authorized data the tool stores is the OAuth token. Deleting the local token file permanently deletes all stored authorized data.
  • When the tool is no longer used, stored tokens are deleted. No other data obtained via YouTube API Services is retained.

6. Contact

For questions about this policy: panteranerow@gmail.com
