1. VideoFactory について

VideoFactory は、開発者(小島一晃 / Ikko Kojima)が自身の YouTube チャンネルに動画をアップロードするために個人的に使用するコマンドラインツールです。一般には配布・提供していません。ユーザーは開発者本人のみです。

2. YouTube API サービスの利用

VideoFactory は YouTube API サービス(YouTube API Services)を利用しています。本ツールを利用することにより、ユーザーは以下に同意したものとみなされます。

  • YouTube 利用規約: https://www.youtube.com/t/terms
  • Google プライバシーポリシー: https://policies.google.com/privacy

3. アクセス・収集するデータ

  • 本ツールは Google OAuth 2.0 を通じて、認可したアカウント自身の YouTube チャンネルへの動画アップロード権限(スコープ: https://www.googleapis.com/auth/youtube.upload)を取得します。将来、自身のチャンネルのデータを分析する機能のために、読み取り専用スコープ(youtube.readonly 等)を追加で取得することがあります。
  • 本ツールが YouTube に送信するのは、開発者が制作した動画ファイルと、そのタイトル・説明文・タグ・サムネイル等のメタデータのみです。
  • 本ツールは、コンテンツの企画・改善のための内部分析を目的として、YouTube Data API から公開データ(自身のチャンネルおよび他の公開チャンネルの動画タイトル・サムネイル・再生数・高評価数・動画URL・公開コメント等)を取得することがあります。
  • 他のユーザーの非公開データや個人情報にはアクセスしません。取得した公開コメント等を、投稿者個人の特定やプロファイリングの目的で利用することはありません。
  • Cookie、アクセス解析、広告技術は一切使用しません。

4. データの保存と共有

  • OAuth 認証トークンは、開発者のローカルPC上にのみ保存されます(外部サーバーには保存しません)。
  • API 経由で取得した公開データも、開発者のローカルPC上にのみ保存されます。保存した API データは、YouTube API サービスのデベロッパーポリシーに従い、保存後30日以内に更新または削除します。
  • 取得した認証情報・アップロードに関する情報・分析用に取得したデータを、第三者に提供・販売・共有・再配布することはありません。
  • 外部へのデータ送信は、Google/YouTube の公式 API エンドポイントに対するもの以外に存在しません。

5. アクセス権の取り消しとデータの削除

  • ユーザー(開発者本人)は、Google のセキュリティ設定ページ https://security.google.com/settings/security/permissions からいつでも本ツールのアクセス権を取り消すことができます。
  • 本ツールが保存するデータは、OAuth トークンと、分析用に取得した公開データのローカルキャッシュのみです。いずれもローカルファイルを削除することで完全に削除されます。
  • 本ツールの利用を終了する場合、保存されたトークンおよび取得済みの API データはすべて削除されます。

English

1. About VideoFactory

VideoFactory is a command-line tool used personally by its developer (Ikko Kojima) to upload videos to the developer's own YouTube channel. It is not distributed or offered to the public. The sole user is the developer.

2. Use of YouTube API Services

VideoFactory uses YouTube API Services. By using this tool, the user agrees to be bound by:

  • YouTube Terms of Service: https://www.youtube.com/t/terms
  • Google Privacy Policy: https://policies.google.com/privacy

3. Data Accessed and Collected

  • The tool obtains, via Google OAuth 2.0, the permission to upload videos to the authorizing account's own YouTube channel (scope: https://www.googleapis.com/auth/youtube.upload). In the future, a read-only scope (e.g. youtube.readonly) may additionally be requested for analyzing the developer's own channel.
  • The only data sent to YouTube are video files created by the developer and their metadata (title, description, tags, thumbnail).
  • For internal content-planning analysis, the tool may retrieve public data via the YouTube Data API (video titles, thumbnails, view counts, like counts, video URLs, and public comments of the developer's own channel and other public channels).
  • The tool never accesses non-public data or personal information of other users, and never uses retrieved public comments to identify or profile individual commenters.
  • No cookies, analytics, or advertising technologies are used.

4. Data Storage and Sharing

  • OAuth tokens are stored only on the developer's local machine (never on external servers).
  • Public data retrieved via the API is likewise stored only on the developer's local machine. In accordance with the YouTube API Services Developer Policies, stored API data is refreshed or deleted within 30 days.
  • No authorized data or retrieved API data is sold, shared, redistributed, or transferred to any third party.
  • The tool communicates only with official Google/YouTube API endpoints.

5. Revoking Access and Data Deletion

  • Access can be revoked at any time via Google security settings: https://security.google.com/settings/security/permissions.
  • The data the tool stores consists only of the OAuth token and a local cache of retrieved public data. Deleting these local files permanently deletes all stored data.
  • When the tool is no longer used, all stored tokens and retrieved API data are deleted.

6. Contact

For questions about this policy: panteranerow@gmail.com
