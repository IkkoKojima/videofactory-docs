VideoFactory について

VideoFactory は、開発者(小島一航 / Ikko Kojima)が自作の教育系動画(参考文献を明示した日本語の教養エッセイ動画)を、開発者自身の YouTube チャンネルへアップロードするためだけに使用する、非公開の個人用コマンドラインツールです。一般への配布・提供・販売は行っておらず、ユーザーは開発者本人のみです。

本ツールは YouTube API サービス(YouTube API Services)を利用しています。データの取り扱いについてはプライバシーポリシーをご覧ください。

YouTube Data API の利用内容

  • videos.insert — 完成した動画ファイルのアップロードと、タイトル・説明文・タグ等のメタデータ設定
  • thumbnails.set — アップロードした動画へのサムネイル設定

認可は Google OAuth 2.0(スコープ: youtube.upload のみ)で行い、対象は開発者自身の Google アカウントに限られます。第三者のデータや他ユーザーのチャンネル・コンテンツには一切アクセスしません。

English Summary

VideoFactory is a personal, internal command-line tool developed and used solely by Ikko Kojima to upload self-produced educational videos to the developer's own YouTube channel. It is not publicly available. The tool uses YouTube API Services and calls only videos.insert and thumbnails.set, authorized via Google OAuth 2.0 with the youtube.upload scope for the developer's own account. See the Privacy Policy and Terms of Service.
