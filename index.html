VideoFactory について

VideoFactory は、開発者(小島一航 / Ikko Kojima)が自作の教育系動画(参考文献を明示した日本語の教養エッセイ動画)を、開発者自身の YouTube チャンネルへアップロードするためだけに使用する、非公開の個人用コマンドラインツールです。一般への配布・提供・販売は行っておらず、ユーザーは開発者本人のみです。

本ツールは YouTube API サービス(YouTube API Services)を利用しています。データの取り扱いについてはプライバシーポリシーをご覧ください。

YouTube Data API の利用内容

  • videos.insert — 完成した動画ファイルのアップロードと、タイトル・説明文・タグ等のメタデータ設定
  • thumbnails.set — アップロードした動画へのサムネイル設定
  • videos.list / channels.list / search.list / playlistItems.list / commentThreads.list — 自身のチャンネルの振り返りと、公開データに基づく市場調査(動画タイトル・サムネイル・再生数・高評価数・公開コメント等の取得)のための内部分析(計画中の機能)

認可は Google OAuth 2.0(スコープ: youtube.upload。分析機能の導入時には読み取り専用スコープを追加予定)で行い、対象は開発者自身の Google アカウントに限られます。分析で取得するのは公開データのみで、他ユーザーの非公開データや個人情報にはアクセスしません。取得データはローカルでの内部分析にのみ使用し、再配布・販売は行いません。

English Summary

VideoFactory is a personal, internal command-line tool developed and used solely by Ikko Kojima to upload self-produced educational videos to the developer's own YouTube channel. It is not publicly available. The tool uses YouTube API Services: it calls videos.insert and thumbnails.set for uploads, and (as a planned feature) read-only endpoints such as videos.list, channels.list, search.list, and commentThreads.list to retrieve public data for internal content-planning analysis. Authorization uses Google OAuth 2.0 for the developer's own account. Only public data is retrieved; it is analyzed locally and never redistributed. See the Privacy Policy and Terms of Service.
